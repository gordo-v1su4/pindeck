## 2024-05-24 - Async Button Feedback
**Learning:** Users lack feedback during anonymous sign-in, which is an async operation.
**Action:** Always wrap async actions in buttons with loading states and disable them.
## 2024-05-24 - Input Accessibility
**Learning:** Inputs using only placeholders are not accessible to all screen readers.
**Action:** Always add aria-label if a visual label is not present.
