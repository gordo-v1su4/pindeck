# Quickstart

Get Pindeck up and running — sign up, upload your first image, and generate an AI variation.

## 1. Create an account

1. Go to [www.pindeck.dev](https://www.pindeck.dev)
2. Sign in with Google or GitHub OAuth
3. Your profile is created automatically on first login

## 2. Upload your first image

1. Click **Upload** or drag and drop an image onto the gallery
2. Add a title, description, and tags
3. Choose a category (e.g., Commercial, Film, Moodboard)
4. The image is stored in Convex and automatically persisted to RustFS

### Upload metadata

Each image supports:

| Field | Description |
|-------|-------------|
| Title | Display name for the image |
| Category | Classification group (Commercial, Film, etc.) |
| Tags | Searchable keywords |
| Group | Project group (e.g., Commercial, Spec Music Video) |
| Project name | Specific project within a group |

## 3. Organize into a board

1. Select images from the gallery
2. Click **Add to Board** → create a new board or add to an existing one
3. Boards support drag-and-drop reordering

## 4. Generate an AI variation

1. Open an image and click **Generate Variation**
2. Choose a variation type:
   - **Shot type** — Changes the camera angle/framing (e.g., close-up, wide shot)
   - **Style** — Applies a visual style transformation
3. The variation is generated via fal.ai and saved as a child image linked to the original

## 5. Create a deck

1. Open a board and click **Create Deck**
2. Select a template
3. Rearrange slides and layouts
4. Present directly from Pindeck
